# host the web site
downloan client from: https://dashboard.ngrok.com/get-started/setup/windows

create your account into ngrok using google.

open ngrok terminal in your desktop.

To test installed or not: ngrok version

To add your account token with client: ngrok config add-authtoken <token created in ngrok account>

To host your app into ngrok clien: ngrok http 8080 or 80 or <application-port>
